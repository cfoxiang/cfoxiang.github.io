母亲节给子女们的4个建议

发表于 2011-05-07

明天是5月8日，母亲节来了，专家们又在嚷嚷国外的母亲节不能承担中国传统文化，宣传什么“中华母亲节”，两个母亲节，对我们大家来说，贵祥以为，其实过哪个并不重要，重要的是我们能用实际行动去为我们的母亲切切实实做一些事情吧。

贵祥的母亲，一直居住在江苏农村，今年因为小天天上学需要人照看，老人家舍不得贵祥花钱雇人，毅然放下农村的几亩地，来到对她来说，梦所不能及的北京，语言的障碍，离乡的孤单，贵祥和芬妹妹只能是多花时间陪陪她了。母亲节，老娘肯定是不知道了，但明天贵祥决定放下手头的工作学习任务，好好让她过个开心的节日。

兄弟姐妹们，大家计划如何呢？贵祥给兄弟姐妹们4个建议吧。

  1. 回家看看。有条件，有时间，带上媳妇和孩子回家看看吧，在这个特别的日子里，让母亲享受天伦之乐。不要找任何理由啊，兄弟姐妹们，人生什么最重要？？亲情！！！
  2. 打个电话。如果您身在异乡，没办法回家，一句简单的爱的问候。“妈妈，节日快乐！”相信不管妈妈知道不知道这个节日，都肯定会开心不已。
  3. 送份礼物。给母亲送件礼物，不求贵重，但要求用心去选择。一束花，一件衣服，甚至一盒母亲爱吃的糕点，……，相信母亲都会幸福很久。
  4. 做顿美餐。这是贵祥计划的部分啊。从小到现在都是母亲给我们做饭，妈妈知道我们爱吃什么，我们知道妈妈爱吃什么吗？好好问问，做一份母亲喜欢的大餐吧，让母亲开心一下。

当然，母亲节别只想到自己的母亲，成家的兄弟们还要考虑一下自己的岳母、姐妹们还要考虑一下自己的婆婆，同时，男人们还是要考虑一下自己的媳妇，怎么说人家也是咱孩子的母亲啊，呵呵。一起按上面的建议做一做啊。

爱是长久的，是永恒的，不只是在母亲节。对我们这些子女来说，每天都应该当母亲节或父亲节为父母做自己力所能及的事。爱他们，不只是语言，更重要的是行动。

兄弟姐妹们，一起从现在起，行动起来吧。

刚才，收到今天手机移动晚报母亲节送礼指南，一并送给兄弟姐妹们吧：

  1. 60后与70后：精心挑选健康食品；回家看看;为父母做一桌佳肴。
  2. 80后：为爹妈拍婚纱照；陪父母去旅游；“脱光”—赶紧找到另一半带回去。
  3. 90后：大声说出对母亲的爱；还可以借助网络工具，写下对母亲的祝福，把孝心传递。
  4. 00后：自己动手做礼物，写上给母亲的祝福。
